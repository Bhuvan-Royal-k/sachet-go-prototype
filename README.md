# Sachet-Go ⚡
**A Full-Stack Micro-Commerce Prototype**

### Problem Statement
Most Quick-Commerce apps (Blinkit/Zepto) focus on high-ticket items and bulk packs. **Sachet-Go** optimizes for the "Bottom of the Pyramid" market, focusing exclusively on ₹1, ₹2, and ₹5 sachets for daily needs.

### Tech Stack
- **Frontend:** React.js (Vite)
- **Backend:** Node.js & Express
- **State Management:** React Hooks (useState, useEffect)
- **Styling:** Custom CSS-in-JS

### Features
- [x] Real-time Search Filtering
- [x] Category-based Sorting
- [x] Dynamic Shopping Cart & Bill Calculation
- [x] Conditional Rendering Checkout Flow
